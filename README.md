
# Relatos de papel

Front for book's e-commerce, development in react and react router v6




## Installation

Requires Node.js v23+ to run.

Install the dependencies and devDependencies and start the server dev.

```bash
  npm install && npm run dev
```


## Features

- Landing Page
- List products
- Detail products
- Cart
- Checkout
